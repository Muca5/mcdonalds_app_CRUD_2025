1. Tela de Adicionar Lanches Favoritos

Esta tela permite que o usuário cadastre seus lanches favoritos.
Ela contém campos para nome do lanche e descrição, além de um botão de Salvar.
Abaixo, é exibida uma lista dos lanches já cadastrados, cada item contendo opções para editar ou excluir o registro.

![Adicionar Lanches](/mnt/data/WhatsApp Image 2025-11-24 at 22.55.00.jpeg)

2. Tela Inicial – Bem-vindo

Após o login, o usuário é recebido com uma mensagem personalizada e um botão de acesso para Gerenciar Produtos, onde poderá visualizar ou editar seus lanches cadastrados.
Um menu no canto superior direito permite navegar pelas demais opções do app.

![Tela Inicial](/mnt/data/WhatsApp Image 2025-11-24 at 22.55.01 (2).jpeg)

3. Tela de Registro de Cliente

Esta é a tela de cadastro do aplicativo, onde o usuário pode criar uma conta informando nome, nickname, e-mail, senha e telefone.
O layout segue a identidade visual vermelha, inspirada na marca escolhida para o tema da atividade.

![Registro](/mnt/data/WhatsApp Image 2025-11-24 at 22.55.01 (1).jpeg)

4. Tela de Login

A tela de login permite que o usuário acesse sua conta existente informando e-mail e senha.
Também há um atalho para a tela de cadastro caso o usuário ainda não possua conta.

![Login](/mnt/data/WhatsApp Image 2025-11-24 at 22.55.01.jpeg)
