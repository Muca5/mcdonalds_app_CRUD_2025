1. Tela de Login

A tela de login permite que o usuário acesse sua conta existente informando e-mail e senha.
Também há um atalho para a tela de cadastro caso o usuário ainda não possua conta.

<p align="center"> <img src="https://github.com/user-attachments/assets/5da1245c-aa2b-40d3-a3f9-fe56a3fb700a" width="300"> </p>
2. Tela de Registro de Cliente

Esta é a tela de cadastro do aplicativo, onde o usuário pode criar uma conta informando nome, nickname, e-mail, senha e telefone.
O layout segue a identidade visual vermelha, inspirada no tema escolhido para a atividade.

<p align="center"> <img src="https://github.com/user-attachments/assets/09d60e16-c8ea-45ea-9198-397627a2e05f" width="300"> </p>
3. Tela Inicial – Bem-vindo

Após o login, o usuário é recebido com uma mensagem personalizada e um botão para Gerenciar Produtos, onde poderá visualizar ou editar seus lanches cadastrados.
Um menu no canto superior direito permite navegar pelas demais opções do app.

<p align="center"> <img src="https://github.com/user-attachments/assets/b0e31935-acb4-4486-8012-3254b5240b7f" width="300"> </p>
4. Tela de Adicionar Lanches Favoritos

Esta tela permite que o usuário cadastre seus lanches favoritos.
Ela contém campos para nome do lanche e descrição, além de um botão de salvar.
Abaixo, é exibida a lista de lanches já cadastrados, cada um com opções de editar ou excluir.
